## Optionality가 왜 핵심인가

| 구분 | 내용 |
|------|------|
| **조건** | GPS 발전소가 LNG·LPG 모두 소화 가능 (세계 최초) |
| **작동 방식** | JKM(LNG 현물가) vs FEI(LPG 현물가) 실시간 비교 → 싼 연료로 발전, 비싼 연료 판매 |
| **수익 예시** | '26년 3월: JKM $20.5 vs FEI $16.2 → Spread $4.33/MMBtu. 1.5 Cargo 실행 → **+300억** 확정 |
| **극단 사례** | '22년 8월 Spread $62/MMBtu → 카고당 3,150억 기회 |
| **Trading과의 시너지** | Physical 전환 외에 Paper 거래(선물·스왑)로도 동일 Spread 수익화 가능 |

## 사업별 1분 카드

| 사업 | 한 줄 핵심 | 수익 Driver | '26 세전이익 |
|------|-----------|------------|------------|
| **LPG** | 프로판·부탄 도매 + Trading | FEI/MB Spread, Optionality | 1,841억 (목표) |
| **GPS 발전** | 세계 최초 LNG·LPG 겸용 1,227MW | SMP × 발전량 + 용량요금(CP) | 1,725억 (목표) |
| **LNG** | 터미널 확장으로 이익 급성장 | 터미널 이용료 + Trading | 143억 → 1,300억 ('31목) |
| **벙커링 EMFS** | IMO 규제 대응 선박 연료 | 급유 마진 | '27~'28 사업 개시 |
| **SKA** | PDH로 프로판→프로필렌 전환 | PDH Spread | 구조개편 검토 중 |
